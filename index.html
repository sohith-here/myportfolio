Haritha Karma Sena

Haritha Karma Sena is a web-based application developed using HTML, CSS, JavaScript, and Python (Django framework) to facilitate efficient waste collection and management. This platform is designed to support the operations of the Haritha Karma Sena, a community-driven initiative in Kerala focused on sustainable waste management. The application enables users to schedule waste pickups, track collection statuses, and manage waste disposal records. It aims to streamline waste collection processes, promote environmental awareness, and enhance community participation in maintaining cleanliness and sustainability.
